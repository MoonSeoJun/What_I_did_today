# 리눅스 시스템 프로그래밍 파일 다루기 기초
## 1. fopen()
c에서 파일에 접근할 수 있게 해주는 함수

fopen() 후 파일 접근이 끝나면 fclose()를 실행해주어야한다.

fopen()에서 지원하는 접근 방법은 "r"(read), "w"(write), "a"(append) 3가지가 있다.

### 1. "r"(read)
"r"(read)는 파일을 읽기 위한 접근 방법이다

파일이 존재하지 않으면 NULL을 반환한다.

"r+"를 사용하면 읽기, 쓰기가 모두 가능하다.

### 2. "w"(write)
"w"(write)는 파일을 작성하기 위한 접근 방법이다.

파일이 존재하지 않으면 새로 파일을 생성한다.

기존에 같은 이름의 파일이 있을 경우, 기존 파일을 삭제 후 새로 파일을 작성한다.

"w+"를 사용하면 읽기, 쓰기가 모두 가능하다.

### 3. "a"(append)
"a"(append)는 파일에 새로 문자를 추가하기 위한 접근 방법이다.

파일이 존재하지 않으면 새로 생성한다.

기본적으로 파일의 마지막 부분에 추가한다.

"a+"를 사용하면 파일의 앞과 뒤를 선택할 수 있다.

---

## 2. fputs()
fopen() 후 파일에 문자를 추가하기 위해 사용되는 함수이다.

사용 방법
```
fputs("String", FILE);
```

---

## 3. ftell()
ftell()은 현재 파일에서의 위치를 반환하는 함수이다.

ftell()의 반환 값의 형태는 long이기 때문에 출력시 %ld를 사용해야 한다.

사용 방법
```
print("%ld",ftell(FILE));
```

---

## 4. fgets()
파일의 문자를 읽어들이는 함수

사용 방법
```
char buf[1024]; 
memset(buf, 0, sizeof(buf)); // buf를 0으로 초기화
fgets(buf, sizeof(buf), FILE); // buf에 파일의 내용이 담김
```
